# SaveMate (Chrome URL Saver Extension)

A simple Chrome extension built using HTML, CSS, and JavaScript that allows users to save URLs manually or directly from the active browser tab.

## Features
- Save any URL by pasting it
- Save the currently active tab
- View saved URLs instantly
- Delete all saved URLs

## Technologies Used
- HTML
- CSS
- JavaScript
- Chrome Extension APIs (Tabs, Storage)

## How to Use
1. Clone the repository
2. Open Chrome and go to chrome://extensions
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select the project folder
